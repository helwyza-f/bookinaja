# Bookinaja Mobile Per-Screen Prompt Blocks For Stitch

Use these blocks as copy-paste prompts in Stitch.

Rule:

- Start with foundations and patterns first.
- Then move to high-priority customer screens.
- Then move to high-priority admin screens.
- Reuse the same visual language across all prompts.

## 1. Foundation Prompt

```text
Design the visual foundation for Bookinaja Mobile.

Bookinaja is a booking and live operations product for gaming centers, entertainment venues, rental spaces, and service businesses.

The design must feel:
- emotional
- engaging
- personal and alive
- premium
- smooth
- delightful
- polished

This must not feel like a generic operational app or SaaS dashboard.

Create:
- color system
- typography system
- spacing system
- radius system
- icon style
- surfaces
- cards
- chips
- buttons
- fields
- live states
- toasts
- centered modals
- bottom sheets

The brand should feel like:
- signal
- pulse
- momentum
- live activity
- premium command

Keep it:
- simple
- clean
- modern
- expressive
- operationally sharp

Less verbose, more icon, more layout, more grid, more feedback, more live.
```
